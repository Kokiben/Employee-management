# Employee Management System

## 📌 Description

This is a backend Employee Management System developed using Spring Boot.
It allows managing employees with basic CRUD operations.

## 🚀 Features

* Add new employees
* View employee list
* Update employee details
* Delete employees

## 🛠️ Technologies

* Java
* Spring Boot
* Maven
* MySQL (if you used it)

## ▶️ How to Run

1. Clone the repo:

```bash
git clone https://github.com/Kokiben/Employee-management.git
```

2. Open in IntelliJ or VS Code
3. Run the Spring Boot application
